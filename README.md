# React Starter Kit v1

## How to use

`Git`을 설치하셨다면, 아래의 명령어를 이용하여 바로 사용하실 수 있습니다.

```
git clone https://github.com/jonggyun/react-starter-kit-v1.git
```

```
yarn start
```

http://localhost:3000 에서 확인하실 수 있습니다.
